Inventarios

Erica Barcenas

Jefe Operaciones Logistica

Contacto: ebarcenasm

Antiguedad: 14 años

El objetivo de inventarios es brindar confiabilidad sobre la mercancía que se posee. Identificando los ajustes que se podrían presentar en la operación como Faltantes, Sobrantes, Siniestros/Robo, Devolución y Donación. Para ello utilizan diferentes procesos como el reporte, Stock check, realizado de manera diaria en el que se compara la información de 3 sistemas SOMS, ECC y EWM.En este reporte se visualizan las diferencias entre SOMS y EWM se encuentran situaciones que son analizadas para cuadrar la diferencia como por ejemplo que no se contabilizara una entrada o salida de mercancía, a veces esto se origina por el proceso que se corre por la noche y algunas piezas son cargadas posterior a este proceso, otros procesos que se utilizan son los inventarios cíclicos y los inventarios totales los cuales consisten en un conteo de la mercancía que está en cantiléver o bloque.

En todo lugar, todos los días, toda la vida.

Logística inversa

Eloisa Villalobos Lopez

Supervisor Operativo D1

Contacto: evillalobos

Antiguedad: 33 años

El propósito del área de logística inversa es disminuir la cantidad de mercancía que se va a saldos.
La responsabilidad para Huehuetoca termina hasta que la mercancía pisa un centro regional a partir de ese momento el centro regional cuenta con 7 días para reportar maltrato, defecto, incompleto, sobrante y faltante. Este departamento visualiza el buzón de cepros se confirma que estén dentro de los 7 días se confirma y se recibe. En el caso de Tultitlan y Tacubaya solo disponen de 24 horas.
Se cuenta con el taller 1 Experto en maderas y el taller 2 maderas salas y colchones. En ciertas ocasiones se opta por utilizar el servicio del proveedor para realizar la reparación.
Cuando se utiliza el servicio del proveedor sale con una orden de servicio en la cual tiene 6 días para confirmar o si no se le puede presentar una devolución cuando confirma el proveedor este tiene 30 días para regresar la mercancía pasado este lapso se puede presentar una devolución si no cumple con el tiempo de entrega se puede presentar una devolución, el proveedor puede solicitar una prórroga donde la primera tiene un cargo del 75% la segunda del 50% y la tercera sin cargo.
Cuando la mercancía ya no se puede salvar se pasa a saldos o donaciones.
Cuando la mercancía es salvada esta se reintegra al inventario por las puertas de recepción

En todo lugar, todos los días, toda la vida.
